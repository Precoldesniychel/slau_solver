# Решение СЛАУ: Метод Гаусса и LU-разложение

Проект реализует численные методы решения систем линейных алгебраических уравнений (СЛАУ). Выполнен сравнительный анализ производительности и точности метода Гаусса (с выбором ведущего элемента и без) и метода LU-разложения. Особое внимание уделено демонстрации преимущества LU-разложения при решении серий систем с одинаковой матрицей.

## Цель работы

Реализация набора функций для решения систем линейных алгебраических уравнений (СЛАУ) методом Гаусса и методом LU-разложения. Сравнительный анализ времени работы алгоритмов и демонстрация преимущества LU-разложения при решении систем с одинаковой матрицей, но разными правыми частями.

## Математические основы

### Метод Гаусса

Метод последовательного исключения неизвестных для приведения матрицы системы к верхнетреугольному виду:

**Без выбора ведущего элемента:**
- Прямое исключение элементов ниже главной диагонали
- Деление на диагональный элемент $a_{kk}$
- Неустойчив при малых значениях на диагонали

**С выбором главного элемента по столбцу:**
- Перед каждым шагом исключения производится перестановка строк
- На диагональ помещается максимальный по модулю элемент столбца
- Обеспечивает численную устойчивость алгоритма

**Сложность:** $O(n^3)$ операций

### LU-разложение

Представление матрицы $A$ в виде произведения нижней ($L$) и верхней ($U$) треугольных матриц:

$$A = L \cdot U$$

где $L$ — нижняя треугольная матрица с единичной диагональю, $U$ — верхняя треугольная матрица.

**Решение системы:**
1. Прямая подстановка: $Ly = b$
2. Обратная подстановка: $Ux = y$

**Преимущество:** разложение выполняется один раз $O(n^3)$, решение для каждой новой правой части — $O(n^2)$

## План экспериментов

### 1. Сравнение времени решения одной системы

Замер времени для размеров $n = 100, 200, 500, 1000$.

**Входные данные:**
- Случайная матрица $A$ (равномерное распределение $[-1, 1]$)
- Случайный вектор $b$

**Измеряемые методы:**
- Гаусс без выбора ведущего элемента
- Гаусс с выбором ведущего элемента
- LU-разложение (суммарное время)
- LU-разложение (отдельно: разложение, подстановки)

**Формат вывода:** Таблица времени (мс)

### 2. Экономия времени при множественных правых частях

Фиксированный размер матрицы $n = 500$.

**Для $k = 1, 10, 100$:**
- Генерация $k$ случайных правых частей $b_1, ..., b_k$
- Замер общего времени решения всех $k$ систем методом Гаусса с выбором ведущего элемента
- Замер общего времени решения всех $k$ систем с использованием LU-разложения (однократное разложение + решения треугольных систем)

**Результат:** Зависимость времени от $k$ (таблица или график)

### 3. Проверка точности на плохо обусловленных матрицах

Для размеров $n = 5, 10, 15$ строится матрица Гильберта $H_{ij} = 1/(i+j-1)$.

**Алгоритм:**
1. Задание точного решения $x = (1, 1, ..., 1)^T$
2. Вычисление правой части $b = Hx$
3. Решение системы каждым методом, получение приближённого решения $\tilde{x}$
4. Вычисление:
   - Относительной погрешности: $\frac{\|\tilde{x} - x\|}{\|x\|}$
   - Невязки: $\|H\tilde{x} - b\|$
5. Сравнение результатов для методов с выбором ведущего элемента и без него

## Установка и запуск

### Требования
- Компилятор C++17 или выше (GCC, Clang, MSVC)
- CMake 3.10+
- Git

### Сборка


git clone <(https://github.com/Precoldesniychel/slau-solver.git)>

cd <slau_solver>

mkdir build

cd build

cmake ..

make

### Запуск
./solver


### Выводы
1. Производительность: LU-разложение показывает сравнимое время с методом Гаусса при решении одной системы
2. Эффективность: При решении систем с одинаковой матрицей и разными правыми частями LU-разложение значительно быстрее (разложение выполняется один раз)
3. Точность: Выбор ведущего элемента критически важен для устойчивости на плохо обусловленных матрицах
4. Численная устойчивость: Методы без выбора ведущего элемента демонстрируют быстрый рост погрешности на матрице Гильберта
